# Pedro16v Tools

## How do I install these formulae?

`brew install pedro16v/tools/<formula>`

Or `brew tap pedro16v/tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "pedro16v/tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
